# $DAI (Dai Stablecoin)

**Documentation Shorthand:** `$DAI`
**Actual Symbol:** `$0x6B175474E89094C44Da98b954EedeAC495271d0F`

## What This Represents

In Uniteum documentation, `$DAI` is a **readable shorthand** for an anchored unit backed by [Dai Stablecoin (DAI)](https://etherscan.io/address/0x6B175474E89094C44Da98b954EedeAC495271d0F).

The actual Uniteum symbol uses the full DAI contract address:
```
$0x6B175474E89094C44Da98b954EedeAC495271d0F
```

## Backing Token

**DAI Contract:** [`0x6B175474E89094C44Da98b954EedeAC495271d0F`](https://etherscan.io/address/0x6B175474E89094C44Da98b954EedeAC495271d0F)

Dai is a decentralized stablecoin generated by the MakerDAO protocol, soft-pegged to the US Dollar. Intended to maintain 1 DAI = $1 USD.

## Anchored Unit Details

**Type:** Anchored (backed 1:1 by real DAI)
**Custodian:** Uniteum contract holding the DAI
**Redeemable:** Yes, burn the anchored unit to retrieve DAI
**Trust Model:** Smart contract custody (unaudited)

## Key Distinctions

| Symbol | Type | Backing |
|--------|------|---------|
| `$0x6B175474E89094C44Da98b954EedeAC495271d0F` | Anchored unit | 1:1 DAI in contract |
| `DAI` (no $) | Symbolic unit | None (just a label) |
| `$DAI` | Documentation shorthand | Refers to anchored version |

## Creating This Unit

```solidity
// Get the "1" token contract
IUnit one = IUnit(0x9df9b0501e8f6c05623b5b519f9f18b598d9b253);

// Create anchored DAI unit
IERC20 dai = IERC20(0x6B175474E89094C44Da98b954EedeAC495271d0F);
IUnit daiUnit = one.anchored(dai);

// daiUnit.symbol() returns: "$0x6B175474E89094C44Da98b954EedeAC495271d0F"
```

## Reciprocal Unit

**`1/$DAI`** (shorthand) = `1/$0x6B175474E89094C44Da98b954EedeAC495271d0F` (actual)

Synthetic unit providing depeg hedge for DAI.

## Example Use Cases

### Stablecoin Diversification

With [$USDC](/tokens/usdc/), [$USDT](/tokens/usdt/), and `$DAI`:

**`$USDC*$USDT*$DAI`** — Three-stablecoin basket:
- Reduces single-point-of-failure risk
- USDC centralized, DAI decentralized → diversified trust model
- If one depegs, others may hold peg

### Decentralized vs Centralized Stable

**`$DAI/$USDC`** — DAI/USDC price ratio:
- Should trade near 1.0 when both pegged
- Captures decentralized vs centralized stablecoin premium
- Arbitrage opportunity when they diverge
