# Guides

Practical how-to guides for Uniteum operations.

## Available Guides

1. **[Creating Units](/guides/creating-units/)** — Create base and compound units
2. **[Forging](/guides/forging/)** — Execute forge operations via Etherscan
3. **[Price Control](/guides/price-control/)** — Influence prices through forging

## Prerequisites

Before following these guides:

- Complete [Getting Started](/getting-started/)
- Have "1" tokens (v0.1) in your wallet
- Understand the [Concepts](/concepts/)

## Tools

These guides use Etherscan for contract interaction. You can also use:

- Direct contract calls via web3.js/ethers.js
- Foundry's `cast` command
- Any wallet with contract interaction support
